# チャッピーへの指示：セクター別騰落率データ取得（全期間）

## 重要：必ずWeb検索して最新データを取得すること

このタスクは「今日時点」のリアルタイムデータが必要です。
学習済みデータや記憶に頼らず、必ずWeb検索で最新の数値を取得してください。
データが古い場合、ブログ記事として使えません。

## 取得日

今日（{作業日の日付}）時点のデータを取得してください。

## データソース（優先順）

1. 株探（kabutan.jp）業種別騰落率ページ
2. Yahoo!ファイナンス 業種別ランキング
3. 日経 業種別指数

※1ソースで全期間揃わない場合は複数ソース組み合わせOK。
※ソース名と取得URLを最終行に明記すること。

## 取得してほしいデータ

東証33業種すべてについて、以下の5期間の騰落率を取得：
- 前日比（%）
- 1週間騰落率（%）
- 1ヶ月騰落率（%）
- 6ヶ月騰落率（%）
- 1年騰落率（%）

## 東証33業種リスト（漏れチェック用）

水産・農林、鉱業、建設、食料品、繊維、パルプ・紙、化学、医薬品、
石油・石炭、ゴム、ガラス・土石、鉄鋼、非鉄金属、金属製品、機械、
電気機器、輸送用機器、精密機器、その他製品、電気・ガス、陸運、海運、
空運、倉庫・運輸、情報・通信、卸売、小売、銀行、証券・商品先物、
保険、その他金融、不動産、サービス

## 出力形式

```で囲んでTSVだけ出力。それ以外一切不要。

```
業種名	前日比(%)	1週間比(%)	1ヶ月比(%)	6ヶ月比(%)	1年比(%)
水産・農林	+0.00	-1.34	+2.50	-5.30	+8.12
鉱業	+1.77	-0.05	...	...	...
...
```

## ルール

- 33業種すべて漏れなく出力
- 数値は小数点第2位まで
- プラスには「+」をつける
- 取得できなかった項目は「N/A」と記入
- 最終行にデータ取得日時とソースURLを明記
- TSV本体以外の説明文は不要（ただしソース情報は必須）
